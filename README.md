# Cartas Zener - Simulación Web

Aplicación web que simula una tirada de 30 cartas Zener aleatorias y calcula automáticamente el número de aciertos

##Tecnologías
-HTML
-CSS
-JavaScript
-PHP

##Funcionalidad 
-Generación aleatoria de cartas
-Cálculo automático de aciertos
-Interfaz web sencilla
